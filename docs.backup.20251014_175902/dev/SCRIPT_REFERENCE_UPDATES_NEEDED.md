## Documentation Template

For each script, add an entry following this template:

```markdown
---

## Script Name

**Script:** `scripts/script_name.sh`  
**Purpose:** Brief description of what the script does

### Synopsis

```bash
./scripts/script_name.sh [OPTIONS] [ARGUMENTS]
```

### Options

| Option | Argument | Default | Description |
|--------|----------|---------|-------------|
| `-h, --help` | - | - | Display help information |
| `--option` | `VALUE` | Default | Description of option |

### What It Does

Detailed explanation of the script's functionality:

1. **Step 1** - What happens first
2. **Step 2** - What happens next
3. **Step 3** - Final steps

### Examples

**Basic usage:**
```bash
./scripts/script_name.sh
```

**Advanced usage:**
```bash
./scripts/script_name.sh --option value argument
```

### Requirements

- Required commands or dependencies
- System requirements
- Configuration requirements

### Exit Codes

- `0` - Success
- `1` - General error
- `2` - Missing dependency
- Other relevant exit codes

### See Also

- Related scripts
- Configuration files
- Documentation references
```
