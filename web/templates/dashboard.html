Virtual Machines
? VM Management:

States:
• Running (green) - VM is on
• Stopped (gray) - VM is off

Actions:
• Start - Power on VM
• Shutdown - Graceful stop (like clicking shutdown)
• Restart - Reboot VM
• Force Stop - Hard power off (use sparingly)

Commands (CLI):
• virsh list --all (list VMs)
• virsh start <vm> (start)
• virsh shutdown <vm> (stop)
• virsh console <vm> (connect)

Professional Tip:
Always use shutdown, not destroy. Destroy is like pulling the power cord - risks data corruption.
